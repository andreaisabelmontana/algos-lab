# algos-lab

Six core algorithms, animated.

**Live:** https://andreaisabelmontana.github.io/algos-lab/

1. Sorting race — bubble / insertion / quick / merge, side-by-side on the same array
2. Pathfinding — BFS / Dijkstra / A*; draw walls with the mouse, drag start/goal
3. Binary search — animated lo / mid / hi pointers
4. BFS vs DFS on a small graph — click any node to start
5. Hash table — chaining vs linear probing, with collision visualisation
6. Backtracking — N-Queens, with attacked-cell highlighting

Plain HTML + canvas. Emerald accent.

Part of the *-lab series: [stats-lab](https://github.com/andreaisabelmontana/stats-lab) · [calc-lab](https://github.com/andreaisabelmontana/calc-lab) · [sql-lab](https://github.com/andreaisabelmontana/sql-lab)
